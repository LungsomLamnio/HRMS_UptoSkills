# 💼 Human Resource Management System (HRMS)

A *web-based HRMS platform* developed using the *MERN Stack* — *MongoDB, **Express.js, **React.js, and **Node.js* — as part of the UptoSkills internship program.

🌐 *Live Preview:* [View Project](https://hrms-upto-skills.vercel.app)

---

## 🚀 Tech Stack

- ⚙ *Backend*: Node.js, Express.js  
- 🖥 *Frontend*: React.js  
- 🗄 *Database*: MongoDB  
- 🌐 *Hosting*: Vercel (Frontend), Render/Atlas (optional for backend/db)

---

## 🎯 Project Goals

The main objective is to simplify and streamline HR operations through:
- 👥 Employee Management  
- 🕒 Attendance Tracking  
- 💰 Payroll Processing  
- 📊 Dashboard & Reporting (upcoming)

This platform aims to provide a *scalable, **intuitive, and **user-friendly* interface for organizations of all sizes.
